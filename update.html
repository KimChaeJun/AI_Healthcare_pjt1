GitHub update

데이터 업데이트 방법

  1. 새 CSV의 컬럼명을 기존과 똑같이 맞춥니다. 예: 나이, BMI, 공복 혈당, 중성 지방, 혈청 크레아티닌, 헤모글로빈, 혈압, label
  2. 프로젝트 폴더의 data/smoking_health_data.csv를 새 파일로 덮어씁니다.
  3. 아래 명령어로 GitHub에 반영합니다.
git add data/smoking_health_data.csv
git commit -m "update dataset"
git push
