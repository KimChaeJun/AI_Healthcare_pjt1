데이터 업데이트 방법

  1. 새 CSV의 컬럼명을 기존과 똑같이 맞춥니다. 예: 나이, BMI, 공복 혈당, 중성 지방, 혈청 크레아티닌, 헤모글로빈, 혈압, label
  2. GitHub 저장소에서 data/smoking_health_data.csv를 새 파일로 교체합니다.
  3. 커밋하면 GitHub Pages 주소의 대시보드가 자동으로 새 데이터를 읽습니다.
  4. 가설별 기준값은 대시보드 화면에서 직접 수정할 수 있습니다.
